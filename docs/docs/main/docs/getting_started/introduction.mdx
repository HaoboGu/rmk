# Introduction

RMK is a high-performance and feature-rich keyboard firmware framework, designed for flexibility and ease of customization. RMK leverages the power of Rust and it's great async ecosystem, provides comprehensive and modern functionality across a wide range of microcontrollers. 

## Features

- **Works on a wide range of microcontrollers**: ESP32, nRF52, STM32, RP2040/2350, CYW(Pi Pico W), etc.
- **Supports both wired and wireless connection**, low-power optimized.
- **Easy configuration** via a single [toml](https://toml.io/) file.
- **Modern keyboard features**: real-time keymap customization, advanced layer support, macro support, tap-hold/tap-dance/morse key support, etc.
- **Supports wired/wireless split keyboard**.
